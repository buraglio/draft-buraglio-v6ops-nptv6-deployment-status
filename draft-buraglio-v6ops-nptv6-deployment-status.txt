               NPTv6 Implementation and Deployment Status
          draft-buraglio-v6ops-nptv6-deployment-status-latest

Abstract

   This draft provides an overview of IPv6-to-IPv6 Network Prefix
   Translation (NPTv6) deployment status.  It lists various NPTv6
   features that have been deployed in production networks.  It also
   provides an overview of NPTv6 implementation and interoperability
   testing status.
